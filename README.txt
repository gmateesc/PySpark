
The code consists of two Python scripts

  o src/01_analysis.py  - preliminary analysis

  o src/02_ETL.py - applies ETL to transform supplier_car.json along the lines to Target_Data.xlsx
                    and stored the resulting CSV file under outputs/


To run the code

o run analysis src/01_analysis.py with

    01_analysis.sh

  Example run is in 01_analysis.log


o run ETL code src/02_ETL.py with

    02_ETL.sh

  Example run is in 02_ETL.log
  The resulting CSV file under outputs and I put a copy under ETL_result
